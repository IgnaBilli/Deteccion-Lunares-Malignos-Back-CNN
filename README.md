# TPO IA - Clasificador de Cáncer de Piel HAM10000

## Instrucciones de Uso

### 1. Descargar el Dataset

Descargar las imágenes y el archivo CSV desde Kaggle:
[Skin Cancer MNIST: HAM10000](https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000?resource=download)

### 2. Preparar los Archivos

Una vez descargado:
- Juntar todas las imágenes de `HAM10000_part_1` y `HAM10000_part_2` en una carpeta llamada `HAM10000_images`
- Colocar la carpeta `HAM10000_images` en el directorio raíz del proyecto
- Colocar el archivo `HAM10000_metadata.csv` en el directorio raíz
- El resto de archivos se pueden eliminar

### 3. API

`appv2.py` es la API que se utiliza para la aplicación web.

### 4. Ejecutar

Correr la API para iniciar el servidor.
